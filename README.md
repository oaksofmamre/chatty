# Chatty

Chatty is a real-time chat app

## Synopsis

Chatty is a real-time chat app

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve

Start the server

```sh
$ node app
```

Access from your browser:

```
localhost:5000
```

### Demo

```
https://sqchat.herokuapp.com
```
